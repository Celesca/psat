# Custom Partner Logos

Drop your custom application partner logos in this directory.

For example, name them `app1.png`, `app2.png`, etc. to match the paths in `components/AppIntegrationSection.tsx`, or update the `appLogos` array in that component if you decide to use different filenames like `garmin.png`.

Recommended format: `.png` or `.svg` with transparent backgrounds, 128x128px or 256x256px.
